# 消費税計算ツール

## 機能

- 税率選択・Enterキー対応の税計算ツール
- 税抜 ⇆ 税込　計算
- 端数処理（切り捨て・切り上げ・四捨五入）
- 消費税率選択（10％/8％）
- 入力値バリデーション

## 使用技術

- HTML
- CSS
- JavaScript (ES6 / class構文)
- DOM操作

## 制作目的

フロントエンドエンジニアとして必要な以下の理解を目的として制作しました。

・DOM操作
・イベント処理
・クラス設計
・実務を想定した計算ロジック実装
